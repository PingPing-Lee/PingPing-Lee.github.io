# 安装nvm、搭建node环境

  1. 安装nvm (opens new window):

  2. 终端输入

    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash

  3. 常见问题：

    1. 若因为网络问题导致安装失败
      在命令行输入，将 nvm node 镜像地址改成国内淘宝镜像

      export NVM_NODEJS_ORG_MIRROR=http://npm.taobao.org/mirrors/node

    2. 出现报错信息如下(一般是新电脑或者重装系统后可能出现的问题):

      Failed to connect to raw.githubusercontent.com port 443: Connection refused

      解决办法请参考链接 (opens new window)

    3. 输入nvm -v会出现-bash: nvm: command not found

      • 进入当前用户的home目录

        cd YouFileName

      • 修改配置文件

        open .bash_profile
<!--  如果文件不存在，自己创建即可，创建命令如下： -->
touch .bash_profile

        直接更改弹出的.bash_profile文件内容,复制一下代码

        export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"

        command + s 保存文件，然后关闭

      • 继续输入以下指令

        source .bash_profile
nvm -v

  4. 使用nvm安装node

    nvm install stable # 安装最新稳定版 node
nvm install 4.2.2 # 安装 4.2.2 版本
nvm install 0.12.7 # 安装 0.12.7 版本

# 特别说明：以下模块安装仅供演示说明，并非必须安装模块
nvm use 4 # 切换至 4.2.2 版本
npm install -g mz-fis # 安装 mz-fis 模块至全局目录，安装完成的路径是 /Users/&lt;你的用户名&gt;/.nvm/versions/node/v0.12.7/lib/mz-fis
nvm use 0 # 切换至 0.12.7 版本
npm install -g react-native-cli #安装 react-native-cli 模块至全局目录，安装完成的路径是 /Users/&lt;你的用户名&gt;/.nvm/versions/node/v4.2.2/lib/react-native-cli

nvm alias default 0.12.7 #设置默认 node 版本为 0.12.7

  5. 常用nvm命令

    • 查看已安装的版本：nvm ls
    • 查看可以安装的版本：nvm ls-remote
    • 安装指定的版本：nvm install version
    • 删除指定的版本：nvm uninstall version
    • 使用选定的版本：nvm use version

# 安装git

  • 安装: brew install git
  • 配置用户名和邮箱：
git config --global user.name "your_name"  
git config --global user.email "your_email@gmail.com"
  • 查看git配置信息：git config --list
  • 对于git不熟悉或者之前使用svn的同学可以看看git教程 (opens new window)

# 常用软件安装

  • VSCode (opens new window) - 为了保证插件统一，最好统一使用VSCode

    • ESLint (opens new window) - 一定要装，编码规范
    • Git Lens (opens new window) - Git相关利器，包括不限于页面内历史操作查看对比等
    • React Native Snippet (opens new window) - 少打一大推代码

  • Sketch (opens new window) - 抠图看样式必备

  • 奇妙清单 (opens new window)- 面板项目管理必备

  • React Native Debugger (opens new window) - 调整样式、模拟Redux dispatch神器

    brew update &amp;&amp; brew cask install react-native-debugger

  • 依据个人喜好安装

    • typora编辑器 (opens new window) -typora编辑器
    • Skim (opens new window) -PDF工具，便于演示
    • SourceTree (opens new window) -拉取仓库代码
    • Iterm2 (opens new window) - 比系统自带好用的终端
    • Oh-my-zsh (opens new window) - 提升终端逼格利器
    • Alfred (opens new window) - Mac 效率工具必备神器
    • 有道翻译 (opens new window) - 翻译咯
    • Alfred-Convert (opens new window) - 方便各种单位换算
    • Colors (opens new window) - 显示各种rgb,hex,hsl颜色
