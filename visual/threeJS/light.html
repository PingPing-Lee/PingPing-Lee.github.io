# 光源类型

# 基础光源

名称 描述
SpotLight(聚光灯光源) 如：灯光、手电筒等
AmbientLight(环境光) 基础光源，作用在当前场景内的所有物体上,改变物体颜色
PointLight(点光源) 空间中的一个点，向所有方向发射光线，如：恒星光源
DirectionalLight(平行光) 所有光线都是平行，如：太阳光照射在地球上

# 特殊光源

名称 描述
HemisphereLight(半球光) 可以参考夕阳和日出时的光，创建更加自然的户外效果
AreaLight(面光源) 散发光线的平面，不是一个点，一整个平面都在散发光线，光线的方向就是反着平面向前推进

# 光源特殊效果

  • LensFlare(镜面眩光)：不是光源，但是可以为光源添加眩光效果

# 聚光灯光源

  • 锥形效果，可以参考手电筒、路灯等光源。（需要设置光源位置）
  • 可以生成阴影

使用方式

new THREE.SpotLight (color, intensity, distance, angle, exponent)

构造器

SpotLight( color : Integer, intensity : Float, distance : Float, angle : Radians, penumbra : Float, decay : Float )

  • color - (可选参数) 十六进制光照颜色。 默认值 0xffffff (白色)。
  • intensity - (可选参数) 光照强度。 默认值 1。
  • distance - 从光源发出光的最大距离，其强度根据光源的距离线性衰减。
  • angle - 光线散射角度，最大为Math.PI/2。
  • penumbra - 聚光锥的半影衰减百分比。在0和1之间的值。默认为0。
  • decay - 沿着光照距离的衰减量。

# 环境光

  • 不需要设置位置，对整个场景内的对象都生效
  • 没有特定的来源，也不会影响阴影的形成
  • 不能作为场景内的唯一光源，需要配合其他光源使用
  • 用来减弱阴影，或者给物体添加一些颜色

使用方式

const ambientLight = new THREE.AmbientLight(0x000000);
scene.add(ambientLight);

构造函数

AmbientLight( color : Integer, intensity : Float )

  • color - (参数可选）颜色的rgb数值。默认值为 0xffffff。
  • intensity - (参数可选)光照的强度。默认值为 1。

# 点光源

  • 单点发光，照射所有方向的光源。（需要设置光源位置）
  • 不会生成阴影

使用方式

const pointLight = new THREE.PointLight( 0xff0000, 1, 100 );
// 在使用时需要设置点光源的位置，不设置会有一个默认值
pointLight.position.set( 50, 50, 50 );
scene.add( pointLight );

构造函数

PointLight( color : Integer, intensity : Float, distance : Number, decay : Float )

  • color - (可选参数)) 十六进制光照颜色。 默认值 0xffffff (白色)。
  • intensity - (可选参数) 光照强度。 默认值 1。
  • distance - 这个距离表示从光源到光照强度为0的位置。 当设置为0时，光永远不会消失(距离无穷大)。默认值 0.
  • decay - 沿着光照距离的衰退量。默认值 2。

# 平行光

  • 平行光是沿着特定方向发射的光。这种光的表现像是无限远,从它发出的光线都是平行的。如太阳光，因为足够远，所以我们认为从太阳发出的光线也都是平行的
  • 会生成阴影

使用方式

const directionalLight = new THREE.DirectionalLight(0xffffff, 0.5);
scene.add(directionalLight);

构造函数

DirectionalLight(color : Integer, intensity : Float)

  • color - (可选参数) 16进制表示光的颜色。 默认值为 0xffffff (白色)
  • intensity - (可选参数) 光照的强度。默认值为1

如果想让平行光产生阴影需要特别设置几项属性：

directionalLight.shadowCameraLeft = -50;
directionalLight.shadowCameraRight = 50;
directionalLight.shadowCameraTop = 50;
directionalLight.shadowCameraBottom = -50;
directionalLight.shadowCameraNear = 2;
directionalLight.shadowCameraFar = 200;
// 更改阴影的模糊值
directionalLight.shadowMapWidth = 4096;
directionalLight.shadowMapHeight = 4096;

# 半球光

  • 光源直接放置于场景之上，光照颜色从天空光线颜色渐变到地面光线颜色,主要模拟室外自然光照
  • 不会生成阴影

使用方式

const hemisphereLight = new THREE.HemisphereLight(0xffffbb, 0x080820, 1);
scene.add(hemisphereLight);

构造函数

HemisphereLight(skyColor : Integer, groundColor : Integer, intensity : Float)

  • skyColor - (可选参数) 天空中发出光线的颜色， 默认值 0xffffff
  • groundColor - (可选参数) 地面发出光线的颜色， 默认值 0xffffff
  • intensity - (可选参数) 光照强度， 默认值 1
